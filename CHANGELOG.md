# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-28

### Added
- Initial project setup with Node.js backend
- Express.js REST API with JWT authentication
- MongoDB integration with Mongoose schemas
- KYC (Identity Verification) system
- Vehicle booking system
- CORS middleware for frontend integration
- Rate limiting for API security
- Helmet.js for security headers
- Express-validator for input validation

### Features
- User authentication (register, login, logout)
- KYC verification gate for bookings
- Vehicle fleet management
- Booking management
- User profile management
- JWT-based session management

## Format

### Types of changes
- **Added** for new features
- **Changed** for changes in existing functionality
- **Deprecated** for soon-to-be removed features
- **Removed** for now removed features
- **Fixed** for any bug fixes
- **Security** in case of vulnerabilities

### How to Add Changes
1. Create a new version header with the date
2. List all changes under appropriate categories
3. Include GitHub issue/PR references where applicable
4. Keep entries concise and user-focused
